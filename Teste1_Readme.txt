Para executar o arquivo siga as instruções abaixo

1. Instale (se ainda não tiver) o editor de texto VSCode
2. Instale (se ainda não tiver) o ambiente Node.js
3. Baixe o arquivo Modulo1Luiz.js e salve em uma pasta à sua escolha
4. Abra o VSCode, clique no menu acima *Abrir*, depois *Abrir pasta* e escolha a pasta onde está salvo o arquivo Modulo1Luiz.js
5. Seleciona o arquivo no menu lateral
6. Abra um terminal dentro do seu Viscose, clique em *Run*, depois *New terminal*
7. No terminal digite o comando ‘*npm install prompt-sync*’ que instalará um recurso para rodar o script 
8. No terminal digite o comando **node** *Modulo1Luiz*.*js* para executar o script
9. Dentro do script você encontrará as instruções de como executar cada algoritmo e o que cada um faz
